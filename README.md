# Ovartaci AR Mirror

**Multimediedesign**
**Udvikler:** Benedicte, clara, Emilia og Laura (2026)

Dette projekt er en interaktiv og immersiv museumsinstallation udviklet til Museum Ovartaci.

Projektet undersøger identitet, transformation og perception gennem et digitalt AR-spejl, hvor besøgende kan opleve forskellige transformationer inspireret af Ovartacis univers.

Formålet med løsningen er at skabe en interaktiv og kropslig museumsoplevelse, der engagerer yngre besøgende gennem digitale transformationer, immersive interaktioner og personlig storytelling.

## Navngivningskonventioner

Projektet anvender konsistente filnavne for at skabe struktur og overskuelighed.

Danske og engelske versioner af siderne adskilles gennem suffixet "-eng".

Eksempler:

- dragen.html
- dragen-eng.html

JavaScript-variabler og funktioner navngives med camelCase, mens CSS-klasser navngives med kebab-case.
